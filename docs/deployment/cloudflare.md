# Deploy to Cloudflare Pages

Deploy your FastSvelte frontend on Cloudflare Pages for lightning-fast global delivery.

## What You'll Use

- **Cloudflare Pages** - Frontend and landing page hosting
- **Backend elsewhere** - Use Azure, Railway, Fly.io, or DigitalOcean for backend
- **Database elsewhere** - Use Neon, Supabase, or another provider

**Note:** Cloudflare Pages is for frontend. Backend needs separate container hosting.

## Cost Estimate

- Cloudflare Pages: **Free** (unlimited sites)
- Plus backend hosting: $10-30/month
- Plus database: $10-20/month

**Total:** ~$Free + $20-50/month for backend/DB

## Why Cloudflare?

- **Global edge network** - 275+ locations worldwide
- **Unlimited bandwidth** - No bandwidth costs
- **Fast** - Served from edge, very low latency
- **Free tier is generous** - Great for side projects
- **DDoS protection** - Built-in security

4. Deploy automatically on push
